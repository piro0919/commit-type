# Commit Type

> VS Code extension to pick a commit type from a preset list.

## ✨ Features

- 📝 Conventional Commits-compatible type picker
- 🎯 Configurable preset types
- ⚡ Quick prompt integration

## 🚀 Development

```bash
pnpm install
# Open in VS Code and press F5 to launch the Extension Development Host
```

## 📦 Installation

Not published to the VS Code Marketplace. Clone and install locally.
